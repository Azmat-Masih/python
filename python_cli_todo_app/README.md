# Rich Todo CLI (In-Memory)

A simple, visually appealing **in-memory CLI Todo application** built with Python and Rich.

All todos exist only during the program runtime â€” once you exit, they are cleared automatically. Perfect for temporary task tracking.

---

## Features

- Add, edit, delete, and list todos.
- Mark tasks as done or undone.
- Search todos by title, note, or tags.
- Clear all tasks in one command.
- Beautiful CLI interface using Rich tables, panels, and styled text.
- Fully in-memory; no files or databases used.
